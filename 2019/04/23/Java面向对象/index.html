对象和类

对象和类的概念

对象是个内存块，里面放相关联的数据和相关方法。
类可以看作是对象的模板，或者图纸，系统根据类的定义来造出对象。
对象我们叫做Object，instance（实例）。类我们叫class。

定义一个类

对于一个类来说，一般有三种常见的成员：属性field、方法method、构造器constructor。这三种成员都可以定义零个或多个。
